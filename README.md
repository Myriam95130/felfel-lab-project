# ARABIZI FOR COLLOQUIAL ARABIC : THE TUNISIAN CASE

In the early 2000s, keyboards were latin keyboards. It was not adapted for north Africa's dialects that come from arabic. North African decided to adapt arabic letters to latin's one and invented a phonetic system based on letters and numbers for non-latin sounds as uvular, emphatic and laryngal ones.

## About this project

The Tunisian dialect has the  particularity of reducing emphatic sounds as 'ط', 'ظ', 'ض', 'ص'; This phenomenon has led to confusions and ambiguities. This project aims to provide solutions, especially for emphatic sounds and their non-emphatic equivalents as 'ظ' and 'ض', 'ت', ط' and 'س', 'ص';

In the first part of our project we chose to keep a default value which are non-emphatic sounds. When the converter encounters an ambiguous character (e.g. 's', 'dh', 't'), it defaults to the non-emphatic equivalent. This means that it can be emphatic or non-emphatic. We introduced this default value to manage ambiguities.
